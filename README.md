# Pre-requisites
- Install [Node.js](https://nodejs.org/en/) version 8.0.0


# Getting started
- Clone the repository
```
git clone  https://github.com/JosephFaraj97/FindYourCar.git FindYourCard
```
- Install dependencies
```
cd FindYourCar
npm install
```
- Build and run the project
```
npm start
